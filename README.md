# Research Library
## 何これ？
これは、色々なアルゴリズムを片っ端から実装するプロジェクトです。
実装言語はC++14を用い、clangでコンパイルでき、追加のライブラリを必要とせず、MITライセンスで提供しています。

## どう使うの？
今のところ、ライブラリを必要なだけあなたのワーキングディレクトリに追加して使う形となっています。

## 何ができるの？
現在までに

- Burrows Wheeler Transform
- Move To Front
- Range Coder
- Fast Fourier Transform

が実装されています。
